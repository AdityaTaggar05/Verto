# Verto - A Skill Sharing Platform

### Official Submission for _Syntax Error'25_ Hackathon
Welcome to Verto, a unique platform designed for individuals who want to share their skills and learn from others in a fun, interactive way. Whether you want to teach something you're good at or learn a new skill, Verto allows you to book one-on-one slots with other users and pay using an in-app currency called "T-Coins". These coins can later be used to customise your in-game avatar, making the learning experience even more engaging!

### Features
- **Skill Sharing:** Teach and learn new skills through one-on-one sessions.
- **Booking System:** Users can book time slots with others for teaching or learning.
- **In-App Currency:** Earn T-Coins by teaching or learning regularly, and use them to customise your avatar.
- **Customisation:** Personalise your avatar with different skins, armours, weapons, and pets.

### Tech Stack
- **Frontend:** Flutter
- **Backend:** GoLang, Fiber
- **Database:** PostgreSQL
- **Authentication:** JWT (JSON Web Tokens)
- **Hosting:** Render
